# The heterogeneous effects of climate policy on households: Evidence from 88 countries

This repository contains code to create all figures and tables in the manuscript "The heterogeneous effects of climate policy on households: Evidence from 88 countries" (https://doi.org/10.1016/j.jeem.2026.103382).

- CPI_1_Collating_Data.R contains code to collate all country-level datasets. Code for creating all country-level datasets including matching tables can be found here: https://github.com/lmissbach/HACCC
- CPI_2_Analysing_Data.R contains code to perform descriptive analyses on the collated dataset, run the boosted regression tree analysis and create all outputs (figures and tables).
